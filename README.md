# ApexPlanet Internship 


## Task 1

## Responsive Multi-Page Website

This project was developed as part of the ApexPlanet Web Development Internship Task 1.

The objective of this task was to create a fully responsive multi-page static website using HTML5 and CSS3 while following modern web design practices.

---

## Project Overview

The website consists of four pages:

- Home
- About
- Services
- Contact

The project focuses on responsive design, clean layout structure, reusable components, and user-friendly navigation.

# 🚀 ApexPlanet Internship - Task 2

## Interactive Web Components Using JavaScript

This project was developed as part of **Task 2** of the ApexPlanet Web Development Internship. The objective was to enhance the existing responsive website by adding interactive and dynamic features using **Vanilla JavaScript**.

---

## 📌 Project Overview

Task 2 focuses on improving user experience through interactive web components and JavaScript functionality.

The website includes:

- Home Page
- About Page
- Services Page
- Contact Page

along with multiple interactive features powered by JavaScript.

---

## ✨ Features Implemented

### 🍔 Hamburger Menu
- Responsive mobile navigation menu.
- Toggle navigation links on smaller screens.

### 🌙 Dark / Light Mode
- Switch between dark and light themes.
- Theme preference stored using Local Storage.

### 🖼️ Image Slider
- Automatic image carousel on the Home page.
- Smooth image transitions.

### 📢 Modal Popup
- Interactive popup window.
- Open and close functionality using JavaScript.

### 📋 Form Validation
- Name validation
- Email validation
- Phone number validation
- Error handling before form submission

### ⬆️ Back To Top Button
- Appears when the user scrolls down.
- Smoothly returns to the top of the page.

### 🔢 Animated Counters
- Dynamic statistics counter animation.
- Used on the About page.

### 🎯 Smooth Scrolling
- Smooth navigation experience across the website.

---

- Flexbox
- CSS Grid

- Local Storage API
